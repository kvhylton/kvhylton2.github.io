HTML Project: Dreaming Places

Project 1 was about creating an informative website. The information is about a place(city, country, or state) that you want to inform people about.

I created a website about Thailand. I chose Thailand because it's somewhere I've never been and it seemed interesting. I've always been interested in learning about the cultutes and history of other places. Thailand is just one of the many places I dream of visiting. HTML and CSS Project: Sales Website

HTML and CSS Project: Sales Website

Project 2 was about creating a sales website. The purpose of this project was to apply the knowledge pf colors, images manipulation, animation, slideshow, and modal window using CSS to create a simple sales page.

I created a website about Xan's Vineyard and Winery. In this project I presented wine for sale. I created a slideshow in the introduction to advertise the product sales, and hero images to present the products. I put three wines and three wine gifts in the product list with a modal window for each.
